# Zhu Chengyi Portfolio

Personal portfolio website for Zhu Chengyi.

Positioning:

Product Design × AI Workflow × Visual Prototyping

## Tech Stack

- Next.js App Router
- Tailwind CSS
- Framer Motion
- TypeScript

## Local Development

```bash
npm.cmd install
npm.cmd run dev -- -p 3000
```

Open `http://localhost:3000`.

## Content Editing

Project, AI Lab, skills, and contact content lives in:

```text
data/site.ts
```

Images are currently represented by placeholder paths such as:

```text
/images/projects/sensebath/web-hero.png
```

Add real files under `public/images/...` when the final visuals are ready.
